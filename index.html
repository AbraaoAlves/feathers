Let your applications fly

Build lightweight, real-time apps in seconds

> npm install feathers

Take Off

Lightweight

No "Framework X" but for Node. Built right on top and with the spirit of Express. Literally replace express with feathers.

Realtime

No real-time abstraction magic. Plain SocketIO and events is all you need to make your web application come to live.

Flexible

You can do everything that Express and SocketIO can do. And services give you the direction to easily structure your data and application logic.

Rapid start

Feathers can get a REST and SocketIO service running in no time. Services are objects that can be used just like an Express middleware. The example on the right creates a simple Todo service that is available as a RESTful service and via SocektIO:
